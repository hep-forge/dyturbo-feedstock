[![Anaconda Platforms](https://anaconda.org/hep-forge/dyturbo/badges/platforms.svg)](https://anaconda.org/hep-forge/dyturbo)

Feedstock for [dyturbo](https://dyturbo.hepforge.org) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

DYTurbo: Fast predictions for Drell-Yan processes including qt-resummation

## Install

```bash
conda install -c hep-forge -c conda-forge dyturbo
```
